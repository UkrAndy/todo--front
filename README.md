# ToDo Tracker — Vue 3 + Vite

## Опис додатку

**ToDo Tracker** — це сучасний багатомовний SPA-додаток для керування особистими завданнями. Додаток дозволяє створювати, редагувати, видаляти та змінювати статус задач, додавати зображення до задач, а також підтримує автентифікацію користувачів, темну/світлу тему, сторінку 404, адаптивний дизайн і зручний інтерфейс.

### Основні можливості:

- Реєстрація та вхід користувача (автентифікація через JWT)
- Перегляд списку задач з фільтрацією за статусом
- Додавання, редагування, видалення задач
- Додавання зображення до задачі (файл, URL, base64)
- Зміна статусу задачі (виконано, в очікуванні, призупинено, скасовано)
- Підтвердження видалення задачі
- Адаптивний дизайн для мобільних і десктопних пристроїв
- Темна та світла тема
- Багатомовність (українська, англійська)
- Сторінка 404 для неіснуючих маршрутів
- Збереження сесії користувача

## Маршрути та API

### Основні маршрути SPA:

- `/login` — сторінка входу
- `/register` — сторінка реєстрації
- `/` — домашня сторінка (тільки для автентифікованих)
- `/todos` — список задач (тільки для автентифікованих)
- `/todos/new` — створення нової задачі
- `/todos/:id` — редагування задачі
- `/:pathMatch(.*)*` — сторінка 404

### API бекенда (Laravel, JWT):

- `POST /api/login` — вхід, повертає токен
- `POST /api/register` — реєстрація
- `GET /api/user` — отримати дані користувача (авторизація через Bearer Token)
- `GET /api/todos` — список задач
- `POST /api/todos` — створити задачу
- `PUT /api/todos/:id` — оновити задачу
- `POST /api/todos/:id/file` — оновити зображення задачі (multipart)
- `DELETE /api/todos/:id` — видалити задачу

### Особливості:

- **Автентифікація:** доступ до задач лише для залогінених користувачів, токен зберігається у localStorage
- **Темна/світла тема:** перемикається вручну, зберігається у localStorage
- **Багатомовність:** перемикається вручну, зберігається у localStorage
- **404:** для неіснуючих маршрутів відображається красива сторінка з підказкою
- **Збереження сесії:** після оновлення сторінки користувач залишається залогіненим (якщо токен валідний)

## Технологічний стек

- **Vue 3** — основний фреймворк
- **Vite** — швидкий білдер і дев-сервер
- **Pinia** — сучасний state-менеджер для Vue
- **Vue Router 4** — маршрутизація
- **Vue I18n** — багатомовність
- **Axios** — HTTP-клієнт для роботи з API
- **JWT** — автентифікація (на бекенді)
- **SCSS/CSS** — стилізація, адаптивність, теми
- **Компонентний підхід** — AppHeader, MainMenu, UserSection, TodoItem, ConfirmDialog, Loader, ThemeSwitcher, LangSwitcher тощо
- **Volar** — рекомендоване розширення для розробки у VS Code

## Запуск та розробка

1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть дев-сервер:
   ```bash
   npm run dev
   ```
3. Бекенд має бути доступний на http://127.0.0.1:8000 (Laravel API)
